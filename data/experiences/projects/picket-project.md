# 백엔드 팀프로젝트 — 실시간 티켓팅 플랫폼 구현

백엔드 서버 엔지니어로서 예매 및 결제 도메인 핵심 기능을 설계 및 구현.
트래픽 증가 대응 및 동시성 문제 해결을 위한 예매 프로세스 구조 고도화.
TossPayments 결제 모듈 연동 및 결제 검증 프로세스 구현.
언어 및 프레임워크는 Java, Spring Boot, Spring WebFlux, Spring Data JPA, QueryDSL 사용.
데이터베이스는 MySQL, Redis 사용.
인프라는 AWS(EC2, RDS, S3, ElastiCache), Docker, GitHub Actions 사용.
테스팅 도구는 Prometheus, Grafana, JMeter, Locust, Swagger 사용.
