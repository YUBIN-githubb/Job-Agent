# 학부 캡스톤 디자인 프로젝트 - RAG를 이용한 의료기기 임상시험 AI 챗봇 개발

성장하는 의료기기 시장에 비해, 인허가 필수 단계인 임상시험계획서 작성은 여전히 어렵고 복잡. 
기존 서비스나 시중의 AI만으로는 심층적인 정보를 얻는 데 한계.
이를 해결하고자 외부의 신뢰할 수 있는 정보를 연동하는 RAG 기술 기반의 AI(aka. MEDIVA)를 구축.
MEDIVA는 최신 규제 정보를 바탕으로 의료기기 임상시험 계획서 작성 가이드를 제공하며 성능 평가 결과 기존 방식보다 훨씬 정확하고 실용적인 지원이 가능함을 확인.

- 팀 리더로서 프로젝트 일정 관리 및 개발 총괄
- 의료기기 인허가 외부 데이터 수집 및 RAG 구현
- 임상시험계획서 작성 효율화를 위한 맞춤형 LLM 챗봇 서비스 로직 설계

태크 스택은 다음을 사용.
- Python
- LangChain
- Streamlit
- OpenAI API
